Imgurder
========

A little commandline utility for uploading to `Imgur`_

The package can be installed once downloaded::

cd <path to download>
cabal install

Cabal will take care of any dependencies

To use::

   $ imgurder <image file path>

.. _`Imgur`: http://www.imgur.com
